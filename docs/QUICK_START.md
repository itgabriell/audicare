# Guia de Início Rápido para Desenvolvedores

Este guia destina-se a desenvolvedores que desejam configurar e executar o módulo de Atendimento Multicanal localmente para testes e desenvolvimento.

## Pré-requisitos

1.  **Node.js e npm:** Certifique-se de ter o Node.js (versão 20 ou superior) e o npm instalados.
2.  **Acesso ao Supabase:** Você precisa das credenciais do projeto Supabase (URL e Chave Anon).
3.  **Repositório Clonado:** O código-fonte do projeto deve estar clonado em sua máquina local.
